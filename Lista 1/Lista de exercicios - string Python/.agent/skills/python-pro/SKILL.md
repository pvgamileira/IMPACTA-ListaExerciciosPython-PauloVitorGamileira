\# Python Pro Skill



Sempre que trabalhar com código Python neste projeto, você DEVE seguir estas regras:



1\. \*\*Tipagem Estática:\*\* Use Type Hints (`typing`) em parâmetros e retornos de funções.

2\. \*\*Código Limpo:\*\* Priorize a legibilidade. O código deve ser elegante e fácil de entender.

3\. \*\*Resolução Didática:\*\* Como este é um ambiente de estudos/exercícios, escreva o código de forma clara, usando as melhores práticas para manipulação de strings.

4\. \*\*Sem enrolação:\*\* Entregue o código focado no que o exercício pede, sem criar arquivos ou estruturas desnecessárias.
